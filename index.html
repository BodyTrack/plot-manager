Plot Manager

A JavaScript wrapper library for the BodyTrack Grapher.

Examples

Example 1
A minimal, non-resizeable plot. [source]
Example 2
An auto-resizing plot which also specifies initial visible ranges for the date axis and the Y axis. You can control whether auto-resizing is enabled by toggling the checkbox. [source]
Example 3
An auto-resizing plot which also constrains the ranges for the date axis and the Y axis. You can control whether the ranges are constrained by toggling the checkboxes. [source]
Example 4
Two auto-resizing plots in separate plot containers, with Y axis labels. [source]
Example 5
Two auto-resizing plots in the same plot container, sharing a Y axis. One plot plot has a custom style. [source]
Example 6
Two auto-resizing plots in the same plot container, each with its own Y axis. One plot has a custom style. [source]
